# Gym App 

Aplicación fullstack para gestión de clases de gimnasio.

## 🚀 Funcionalidades
- Registro y login (JWT)
- Reservas de clases
- Panel de administrador
- Crear y eliminar clases
- Control de cupos

## 🛠 Tecnologías
- React
- Node.js
- Express
- MongoDB

## ▶️ Cómo correr el proyecto

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start
